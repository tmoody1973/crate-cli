# Documentation

This is the canonical docs index for Crate.

If you are editing docs, start here:

- Use the files in this directory root as the source of truth.
- Treat `docs/archive/` as historical drafts, not current product or architecture docs.
- Treat `docs/plans/` as working design notes.
- Treat `docs/marketing/` as launch and positioning collateral.
